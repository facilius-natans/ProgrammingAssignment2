Human Activity Recognition Using Smartphones Dataset

Contains only the measurements on the mean and standard deviation 
for each measurement.

The dataset includes the following files:

- README.txt 

- Code_book.txt: describes strructure and variables of the tidy_dataset

- run_analysis.R : script that allows to transform the large source file into tidy_dataset 

- tidy_dataset.txt: resulting fille containing  only the measurements on the mean and 
   standard deviation for the required variables
